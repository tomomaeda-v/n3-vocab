# N3単語アプリ セットアップ手順書

ベトナム人学習者向け JLPT N3 単語アプリです。
- 管理者：登録者全員の進捗をダッシュボードで管理
- 登録者：自分だけの進捗を確認
- 日本語の発音（ブラウザ音声合成）
- フラッシュカード／4択クイズ／苦手単語の復習／学習カレンダー
- URLを発行して、管理者・候補者にWebでシェア

収録単語数：**約1,800語（N3レベル）／ 日本語・読み・ベトナム語訳**

---

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面 |
| `app.js` | アプリのロジック |
| `vocab.js` | 単語データ（約1,800語） |
| `config.js` | **あなたのFirebase設定と管理者メールを記入** |
| `firestore.rules` | データベースのセキュリティルール |

## まず「デモ版」で動作確認（Firebase不要・1分）

Firebaseを設定しなくても、その場で試せます。

1. 5つのファイルを同じフォルダに置く
2. `index.html` をブラウザで開く（ダブルクリック）
3. 「Đăng ký（登録）」からメール・パスワードを適当に作成 → ログイン

> デモ版では進捗はそのブラウザ内（localStorage）にだけ保存されます。
> 別の人・別の端末とは共有されません。**全員の進捗を共有・管理したい場合は、次のFirebase設定が必要です。**
> デモで管理画面を見たい場合は、`config.js` の `ADMIN_EMAILS` に登録したメールでログインしてください。

## 本番版：Firebaseを設定して「URLでシェア」する

複数のユーザーの進捗を管理者が一覧管理するには、共有データベース（Firebase）が必要です。**すべて無料枠で十分**動きます。

### ① Firebaseプロジェクトを作成
1. https://console.firebase.google.com/ にアクセス（Googleアカウントでログイン）
2. 「プロジェクトを追加」→ 名前（例：`n3-vocab`）→ 作成

### ② アプリ（Web）を登録して設定値を取得
1. プロジェクト画面で **「</>」（ウェブ）アイコン** をクリック
2. アプリ名を入力 →「アプリを登録」
3. 表示される `firebaseConfig` の値（apiKey など6項目）をコピー
4. `config.js` を開き、`FIREBASE_CONFIG` の各 `"ここに…"` を貼り替える

### ③ 認証（ログイン）を有効化
1. 左メニュー「構築 → Authentication」→「始める」
2. 「ログイン方法」タブ → **「メール/パスワード」を有効化** → 保存

### ④ データベースを作成
1. 左メニュー「構築 → Firestore Database」→「データベースの作成」
2. ロケールは任意（例：asia-northeast1）→ **「本番環境モード」** で開始
3. 作成後「ルール」タブを開き、`firestore.rules` の中身を貼り付けて「公開」
   - ※ ルール内の `admin@example.com` を、あなたの管理者メールに書き換える

### ⑤ 管理者メールを設定
`config.js` の `ADMIN_EMAILS` に管理者のメールを入れる（複数可）。
**`firestore.rules` 内のメールと必ず一致**させること。

```js
const ADMIN_EMAILS = ["tomo.maeda720@gmail.com"];
```

### ⑥ 公開してURLを発行（Firebase Hosting・無料）
ターミナル（コマンドプロンプト）で、ファイルのあるフォルダに移動して実行：

```bash
npm install -g firebase-tools     # 初回のみ
firebase login                    # ブラウザでGoogleログイン
firebase init hosting             # 公開フォルダ=「.」/ SPA=No / 既存プロジェクトを選択
firebase deploy                   # 公開！
```

完了すると次のようなURLが発行されます：
`https://n3-vocab.web.app`

このURLを **管理者・候補者にシェア** すればOK。
- 候補者：自分でメール登録 → 自分の進捗が見える
- 管理者：`ADMIN_EMAILS` のメールでログイン →「Quản trị（管理）」タブで全員を一覧

> Hostingを使わず、Netlify・Vercel・GitHub Pages など他のホスティングに5ファイルを置いてもOKです。

## 使い方（候補者向け・ベトナム語UI）

| タブ | 内容 |
|---|---|
| Trang chủ | 進捗・統計・学習カレンダー |
| Thẻ từ | フラッシュカード（全単語／苦手のみ） |
| Trắc nghiệm | 4択クイズ（日→越／越→日、苦手を優先出題） |
| Danh sách | 全単語の検索・絞り込み |

- 🔊 ボタンで日本語の発音を再生（端末の日本語音声を使用）
- クイズやカードで「✅/❌」を押すと進捗・苦手判定が自動更新されます
- 同じ単語に3回連続で正解すると「習得（mastered）」になります

## よくある質問

**Q. 発音が鳴らない**
→ 端末に日本語の音声エンジンが必要です。多くのPC/スマホは標準で対応。
iPhone/Macは標準対応、Windows/Androidは日本語言語パックの追加で改善します。
ユーザーが一度画面をタップした後に再生されます（ブラウザの仕様）。

**Q. 管理画面が出ない**
→ ログインしたメールが `config.js` の `ADMIN_EMAILS` と一致しているか確認。
本番版では `firestore.rules` のメールも一致が必要です。

**Q. 単語を追加・修正したい**
→ `vocab.js` の配列に `["日本語","よみ","nghĩa tiếng Việt"]` を追加して再デプロイ。

**Q. 無料で足りる？**
→ 数百人規模の学習・進捗保存ならFirebase無料枠（Spark）で十分です。
